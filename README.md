# AutoSubGen-Offline

AutoSubGen-Offline is a fully offline desktop application that automatically generates subtitles from audio and video files using OpenAI Whisper, PyTorch, and FFmpeg.

The application is designed for Windows users who want a simple, reliable, and internet-free subtitle generator with a graphical user interface.

---

## Features

- Fully offline transcription (no internet required)
- Supports audio and video files (MP3, WAV, MP4, MKV, etc.)
- Multiple language support via Whisper
- Powered by OpenAI Whisper
- Uses FFmpeg for audio extraction
- Desktop graphical user interface
- Portable executable build
- Designed for future batch processing support

---

## Tech Stack

- Python 3.10+
- OpenAI Whisper
- PyTorch
- FFmpeg
- PyQt or Tkinter
- PyInstaller
